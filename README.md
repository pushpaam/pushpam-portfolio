# Pushpam Kumar — Portfolio

A modern, animated portfolio website built with React, TypeScript, Vite, Tailwind CSS, and Three.js.

🌐 **Live Site:** [https://YOUR_USERNAME.github.io/pushpam-portfolio](https://YOUR_USERNAME.github.io/pushpam-portfolio)

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 6** (build tool)
- **Tailwind CSS v4**
- **Three.js** + **@react-three/fiber** (3D animations)
- **Motion** (Framer Motion v12)

## Features

- Animated Hero with flying role text
- 3D typography in About section
- Interactive Gallery with flip cards
- Contact section with one-click email copy
- Admin panel to customize content (login: `admin` / `pushpam@2009`)
- Dynamic background color themes
- Fully responsive

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Deploy

This project auto-deploys to GitHub Pages on every push to `main` via GitHub Actions.

### Manual Setup
1. Go to **Settings → Pages** in your GitHub repo
2. Set **Source** to **GitHub Actions**
3. Push to `main` — the workflow handles the rest
